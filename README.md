# CodeGuard – Online Code Plagiarism Checker

## 📌 Overview
CodeGuard is a full-stack web application that detects code plagiarism using AST (Tree) analysis and hashing techniques. It identifies logical similarities even when variable names or formatting are changed.

## 🚀 Features
- AST-based comparison (Tree)
- Hash-based similarity detection
- Similarity % with risk levels
- Interactive demo UI

## 🧠 Data Structures Used
- Tree (AST)
- Arrays / Lists
- Hash Tables
- Graph (visualization)

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
