System Architecture

About the technologies powering CodeGuard.

CodeGuard is a full-stack system designed to send data from this Vanilla JS frontend to a Python Flask server. The server parses Python files into Abstract Syntax Trees (AST) using the built-in ast module. This effectively ignores variable names and formatting, whilst simultaneously generating Winnowing k-gram hashes. A combined score is calculated and matching structural lines are returned to the UI for visualization.
